# 📱 Telegram Mini App — Productivity Suite

## ✨ Описание
Мини‑приложение для Telegram, объединяющее управление задачами, Pomodoro‑таймер, мини‑игры и AI‑чат.  
Открывается внутри Telegram по кнопке бота и предоставляет удобный интерфейс для продуктивной работы и отдыха.

---

## 🛠️ Стек технологий
- 📄 HTML5 — структура экранов
- 🎨 CSS3 — адаптивный дизайн, переменные для тем
- ⚙️ JavaScript ES6 — логика приложения, работа с DOM
- 📦 ES6 Modules — разделение кода (`api.js`, `task.js`, `app.js`, `theme.js`)
- 💾 SQLite — хранение задач и логов (task, task_log)
- 🌐 Fetch API — асинхронные запросы к серверу
- 🤖 Telegram WebApp API — интеграция с Telegram Mini App
- 🚀 FastAPI — серверная часть для REST‑эндпоинтов `/api/tasks`
- 🌍 GitHub Pages — публикация фронтенда
