📱 Telegram Mini App — To Do List
🚀 Описание
Мини‑приложение для Telegram, объединяющее управление задачами и развлекательные модули.
Приложение открывается внутри Telegram по кнопке бота и предоставляет удобный интерфейс для работы с задачами, таймером Pomodoro, мини‑играми и AI‑чатом.

🛠️ Стек технологий
HTML5 — структура экранов.

CSS3 — адаптивный дизайн, переменные для тем, стилизация кнопок и карточек.

JavaScript ES6 — логика приложения, работа с DOM, фильтры задач, отправка форм.

Modules — разделение кода (api.js, task.js, app.js, theme.js).

SQLite — хранения задач и логов выполнения (task, task_logs).

Fetch API — асинхронные запросы к серверу.

Telegram WebApp API — интеграция с Telegram Mini App (window.Telegram.WebApp).

FastAPI — серверная часть для REST‑эндпоинтов /api/tasks.

GitHub Pages — публикация фронтенда и тестирование.

Установка и запуск:

Склонируйте репозиторий:
git clone https://github.com/muhammedaminazat-a11y/Telegram_web_app.git

Перейти в папку проекта:
cd frontend

Запустить локальный сервер:
python -m http.server 5500

Открыть в браузере:
http://127.0.0.1:5500/frontend/js/screens/index.html
